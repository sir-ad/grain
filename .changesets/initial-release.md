# Grain - Initial Release

Universal interaction layer for AI interfaces.

## What's Included

- `grain` - Core parser, validator, state machine
- `@grain/web` - Web adapter with semantic HTML
- `@grain/cli` - Terminal adapter
- `@grain/mcp` - MCP protocol adapter
- `@grain/mcp-server` - MCP server

## Install

```bash
npm install grain
npm install @grain/web
```
