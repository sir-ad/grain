# Grain - Initial Release

Universal interaction layer for AI interfaces.

## What's Included

- `grain` - Core parser, validator, state machine
- `@grain.sh/web` - Web adapter with semantic HTML
- `@grain.sh/cli` - Terminal adapter
- `@grain.sh/mcp` - MCP protocol adapter
- `@grain.sh/mcp-server` - MCP server

## Install

```bash
npm install grain
npm install @grain.sh/web
```
