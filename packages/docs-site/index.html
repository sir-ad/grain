Meridian Layer Architecture

Grain operates at the Meridian Layer—the intersection of model output and interface rendering. It decouples what an agent wants to do from how it is presented.

<stream>

Handles partial content and streaming states. Supports speed, mode, and cursor attributes.

<think>

Encapsulates internal reasoning steps. Can be made visible="true|false" based on UI preference.

<tool>

Formalizes tool execution. Attributes: name, args, status (pending, running, success, error).

<artifact>

Persistent document blocks. Ideal for code, markdown, or rich data visualizations.
