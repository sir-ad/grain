# `@grain.sh/mcp-server`

Stdio MCP server for Grain tooling.

## Install

```bash
npm install -g @grain.sh/mcp-server
```

## Usage

```bash
grain-mcp
```

The server exposes these built-in tools:

- `grain.parse`
- `grain.validate`
- `grain.primitives.list`
- `grain.message.create`

Docs: https://sir-ad.github.io/grain/
