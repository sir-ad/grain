# `@grain.sh/create-grain-app`

Scaffolding tool for creating a Grain application with a working starter template.

## Usage

```bash
npx @grain.sh/create-grain-app@latest my-ai-app
```

The package also exposes these binaries:

- `create-grain-app`
- `create-grain`

Current template:

- `vite` for vanilla web components

Docs: https://sir-ad.github.io/grain/
